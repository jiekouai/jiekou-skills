# 接口AI Skills

接口AI 官方技能仓库，托管可复用的 `SKILL.md` 技能文件，适配主流 Agent 生态系统。

## 安装技能

运行以下命令一键安装：

```bash
npx skills add jiekouai/jiekou-skills --skill jiekou-docs
```

将 `--skill` 替换为下方「可用技能」中的任意名称。安装完成后，如需生效请重启 Agent 运行时。

## 可用技能

| 技能名称 | 描述 |
| --- | --- |
| `jiekou-docs` | 接口AI 平台文档与 API 集成参考 |

## 参与贡献

1. Fork 本仓库并创建新分支。
2. 在 `skills/<skill-name>/` 目录下新增或更新技能文件。
3. 提交 PR，描述清楚使用场景、触发词及验证步骤。
4. 审核通过后自动合并。
